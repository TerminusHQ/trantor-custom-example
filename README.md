# Trantor 自定义控件与容器示例

### 运行
```shell
# 指定 npm 源
npm config set registry  https://registry.npm.terminus.io

# 全局安装t-tools 
npm i @terminus/t-tools -g
# OR
yarn add @terminus/t-tools -g

## 运行
yarn dev
```
